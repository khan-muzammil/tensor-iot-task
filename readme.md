## Instructions

- This project is built using the following
  - Node version - 18.3.0
  - Vitejs - React app
  - Axios
  - Chakra UI
  - React-paginate
  - Tailwind css
  - Typescript
  - React Router
- To get started with this, clone the repo, `cd` into the `tensor-iot-task`
- Run the following command `npm i` or `yarn`
- Then `npm run dev` This should open up the app on `localhost:5173`
- All the cases mentioned have been covered, here is a sample url to test out the shareable url `http://localhost:5173/?filter=success&start=2022-06-12&end=2023-06-12`
